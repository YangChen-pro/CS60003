# 需求说明

## 目标

在 `hw1/` 目录内完成 HW1 作业代码，实现 EuroSAT 三层 MLP 分类器。

## 已确认范围

- 手写前向传播、损失函数、反向传播、SGD、学习率衰减、L2 正则
- 支持至少两种激活函数切换
- 支持数据加载、训练、测试评估、超参数搜索
- 生成训练曲线、混淆矩阵、第一层权重可视化、错例图
- 生成简写实验报告，至少包含必要指标、图表和结论
- README 需要说明依赖和运行方式
- 完成后推送到 GitHub，再在远端 `/data/yc/CS60003` 拉取并运行验证
- 在满足作业要求前提下持续优化三层 MLP 的极限性能

## 约束

- 不使用 PyTorch / TensorFlow / JAX 的自动微分能力
- 代码简单、注释适量、工程化但不过度
- 所有 HW1 相关代码放在 `hw1/`
- 最终提交主模型必须仍然是三层 MLP
- 训练后端固定为远端 CuPy
